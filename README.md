# 🚀 Node.js Health API

A minimal REST API built using Express.js that provides a health check endpoint.

## 📦 Installation

```bash
npm install
```

## ▶️ Run the Server

```bash
npm start
```

Server will run on:   http://localhost:8080

## 🔍 API Endpoint

### GET /health

Returns the status of the server.

**Response:**

```json
{ "status": "OK" }
```

## 🛠 Tech Stack

* Node.js
* Express.js

## 📌 Notes

* Port: `8080`
* `node_modules` is ignored using `.gitignore`
